# WMS Logística

Sistema de gerenciamento de estoque desenvolvido com Google Sheets e Google Apps Script.

## Funcionalidades

- Controle de estoque
- Busca inteligente por SKU
- Picking Inteligente
- Dashboard operacional
- Navegação por ruas
- Automação de processos logísticos

## Tecnologias

- Google Sheets
- Google Apps Script
- JavaScript

## Objetivo

Desenvolver uma solução para otimizar processos logísticos e aumentar a eficiência operacional.

## Principais aprendizados

- Automação de processos
- Desenvolvimento de soluções de negócio
- Manipulação de dados
- Lógica de programação
- Desenvolvimento com Google Apps Script
